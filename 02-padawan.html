Authentication

In order to execute API operations, including GETting or POSTing a resource, you need to authenticate first


After providing your credentials, you get back a token


You then send this token value in a X-Auth-Token header for every subsequent HTTP request

HTTP Request


POST /v2.0/tokens
Host: identity.api.rackspacecloud.com
Content-Type: application/json

{
  "auth": {
    "passwordCredentials": {
      "username": "MY_USERNAME",
      "password": "MY_PASSWORD"
    }  
  }
}
