What is cloud?

Carol Von Canon - Santa Gertudes Cattle
Flickr

Not so much...

  • Distributed hardware across the globe
  • not just 1 server
  • Infrastructure is manipulated by APIs
  • not by a single hypervisor
  • Loads of cool services for different workloads
  • not just a basic Operating System
  • Horizontal scalability
  • not the option to add expensive hardware

What is OpenStack?

  • Open-source software project
  • Modules that provide cloud infrastructure
  • Written in Python: github.com/openstack
  • 100% free!
  • Community driven and independently led

Where does it come from?

Created in 2010 out of a partership
between Rackspace and NASA

Needed a rapidly scalable software system
that could run clouds on standardized hardware

"Wow, you just wrote the code we were going to write."

Since September 2012, it's being run
by its own independent foundation
